# Contributing to ABT-CLI 🚀

Thank you for your interest in contributing! ABT-CLI is currently in **BETA**, and we welcome all types of contributions.

## 🛠 Branching Strategy
- **`main`**: Stable production-ready code.
- **`Beta`**: Release candidates for testing.
- **`Dev`**: Active development and integration.
- **`On_Dev`**: Cutting-edge features and experimental work.

## 🤝 How to Contribute
1. **Fork** the repository.
2. Create a feature branch from `Dev`.
3. Ensure your code passes all **TypeScript** builds (`npm run build`).
4. Perform a test generation to verify the Android structure.
5. Submit a **Pull Request** to the `Dev` branch.

## 📜 Code Style
- Use TypeScript (ESM).
- Follow the modular architecture (`src/commands`, `src/utils`).
- Ensure all new features are documented in the `README.md`.
