# Security Policy 🛡️

## Supported Versions
Only the latest **BETA** version is currently supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.2.0   | :white_check_mark: |
| < 1.2.0 | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it via the following steps:
1. Do not open a public issue.
2. Email the maintainers (or contact via GitHub private reports if available).
3. Provide a detailed description and steps to reproduce.

We aim to respond within 48 hours and provide a fix as soon as possible.
